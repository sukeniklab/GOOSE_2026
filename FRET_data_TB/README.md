# Trevor's in-cell FRET data
* ```cells_concat_raw.csv``` - raw in-cell dataset concatenated from experimental data from image analysis pipeline ran on microscope images
* ```cells_concat_filtered.csv``` - filtered in-cell dataset produced by dfMake.ipynb
* ```constructs_df.csv``` - aggregated data per-construct produced from cells_concat_filtered data, also in dfMaker.ipynb
* ```invitro_conditions.csv``` - data from constructs in-vitro. Folded into constructs_df.csv in dfMaker.ipynb
* ```dfMaker.ipynb``` - filters raw cell dataset to produce figuremaking data
* ```ViolinPlots.ipynb``` - function to create comparative violin plots
* ```basal-delta_ef.ipynb``` - Graphmaking scripts for Ef delta plots
* ```Loc-vs-Ef.ipynb``` - Graphmaking and analysis scripts for localization and localization changes
* ```Figure_graphs``` - Folder which contains figure plot SVGs
